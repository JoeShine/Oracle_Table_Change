项目信息
───────────────────────────────────────────────────────────────────────────
项目名称：OracleBatchUpdater
项目类型：C/S架构桌面应用程序
目标平台：Windows 7/8/10, Windows Server 2008/2012/2016
编程语言：Python 3.8+
GUI框架：tkinter
数据库驱动：oracledb
打包工具：PyInstaller

项目位置
───────────────────────────────────────────────────────────────────────────
主目录：/workspace/oracle_updater
源码目录：src/
配置文件：config.json
日志目录：logs/

模块说明
───────────────────────────────────────────────────────────────────────────
1. config_manager.py  - 配置管理（连接、配置保存）
2. logger.py          - 日志管理（记录、导出）
3. db_connection.py   - 数据库连接（连接、查询、执行）
4. excel_handler.py    - Excel处理（读取、验证、导出）
5. data_updater.py     - 数据更新（备份、更新、清理）
6. gui.py             - 图形界面（所有用户交互）

使用方式
───────────────────────────────────────────────────────────────────────────
方式1：运行Python源码
  $ pip install -r requirements.txt
  $ python main.py

方式2：打包为exe
  Windows: 运行 package.bat
  或: pyinstaller OracleBatchUpdater.spec --clean

方式3：验证安装
  Windows: 运行 verify.bat
  或: python test_all.py

部署说明
───────────────────────────────────────────────────────────────────────────
1. 打包exe后，将dist目录下的所有文件复制到目标电脑
2. 配置Oracle Instant Client（如果未安装Oracle客户端）
3. 直接运行OracleBatchUpdater.exe
4. 无需安装Python环境

下一步
───────────────────────────────────────────────────────────────────────────
1. 在Windows环境中运行 package.bat 打包
2. 将打包后的文件部署到目标电脑
3. 配置Oracle客户端
4. 运行程序并测试
